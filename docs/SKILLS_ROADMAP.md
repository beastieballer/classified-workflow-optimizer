## Priority Custom Skills to Build Next (Recommended)

### 1) Mission-Control Operator
**Goal:** One-command update loop for Idea Vault, Action Board, Build Tracker, Activity Log, and deck packaging.
- Standard update templates
- Auto-log entry format
- “daily sweep” mode and “quick refresh” mode

### 2) Deck Forge
**Goal:** Generate polished one-pagers/slides/PDFs from a single concept brief.
- Clean style presets (exec, investor, technical)
- Print-safe exports
- File naming/versioning convention

### 3) Security Ops Lite (Mac mini)
**Goal:** Repeatable weekly security check + concise risk report.
- OpenClaw audit + update status + exposure summary
- Risk score output (low/med/high)
- Fix-plan checklist (critical first)

### 4) Outreach Pack Builder
**Goal:** Turn a concept into outreach-ready assets in one run.
- ICP bullets
- 5-message outreach sequence
- Discovery call script
- Pilot offer sheet

### 5) Session Checkpoint+Handoff
**Goal:** Standardized end-of-session executive handoff.
- Capture decisions, shipped outputs, open loops, next action
- Optional short “next session starter prompt”
